This repository is not one single project. It is a collection of different projects to get to grips with XGBoost.
The first project is gender classifier based on the Data Analytics Lab video on XGBoost https://www.youtube.com/watch?v=wqnwm15bg7A. 
The second project is a NIDS classifier following the kaggle tutorial from Pranav J https://www.kaggle.com/code/pranavjha24/cic-ids-2017-xgboost-classification.
The NIDS uses the CIC-IDS-2017 datasets.
